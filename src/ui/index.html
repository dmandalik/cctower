Live sessions ?

Every open Claude Code chat and its state: green working · yellow needs your input · red hit an issue · blue done. The bell mutes that project's notifications. Chats idle for 4+ hours drop off.

Controls ?

Global settings — every chat picks them up on its next turn. Mode: observe logs silently, advise shows pre-flight hints, gate blocks heavy prompts when context/quota crosses your thresholds (resend with !force to override).

Quota ?

Your Claude usage. official = rate-limit numbers from Claude Code itself (when it exposes them). local estimate = tokens counted from your own transcripts over rolling 5-hour and 7-day windows.
Landing cards ?
A receipt for each finished turn: what changed, what actually ran, and whether Claude's claims were checked. VERIFIED = a test/build ran and passed. UNVERIFIED = work done but nothing proved it. FAILED = a run failed. NO-OP = nothing changed.
